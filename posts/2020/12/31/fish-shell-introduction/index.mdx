## Install Fish shell

Linux and Windows Subsystem Linux

```bash
sudo apt -y install fish
```

macOS

```bash
brew install fish
curl -L https://get.oh-my.fish | fish
sudo bash -c 'echo /usr/local/bin/fish >> /etc/shells'
chsh -s /usr/local/bin/fish
```
